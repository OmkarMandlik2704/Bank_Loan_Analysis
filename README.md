# Bank_Loan_Analysis
The Bank Loan Analysis Project determines loan eligibility for street vendors using weekly transaction data. It applies data cleaning, outlier removal, and Random Forest Classification to classify vendors. The model achieves high accuracy, aiding in better loan decisions. Feature importance analysis helps optimize loan approvals and risk assessment.

1.Key Objectives
Analyze weekly earnings to determine loan eligibility
Remove outliers using statistical methods
Train a machine learning model for loan classification.

2.Tech Stack
Python – Data processing and modeling
Pandas & NumPy – Data manipulation
Scikit-learn – Machine learning model training
Matplotlib – Data visualization.

3. Data Insights
Top eligible and non-eligible vendors based on earnings
Distribution of loan approvals and rejections
Impact of weekly earnings variability on loan eligibility.

4. Model Performance
Trained using Random Forest Classifier
Evaluated with accuracy, F1-score, recall, and precision
Feature importance analysis for better lending strategies.

5. Future Enhancements
ML-based risk assessment for better loan allocation
Predicting optimal loan repayment periods.
